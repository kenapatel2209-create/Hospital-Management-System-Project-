# Hospital-Management-System-Project

## 1.Project Overview

-The Hospital Management System is a console-based application developed using C++ and Object-Oriented Programming (OOP) concepts. It helps manage hospital operations such as doctors, patients, staff, appointments, and billing efficiently.

-This project demonstrates core OOP principles including Encapsulation, Inheritance, Polymorphism, and Abstraction.

(i)Features Doctor Management :

-Add new doctors View all doctors Store specialization, availability, and consultation fee

(ii) Patient Management :

-Register new patients View patient details Search patient by ID Assign doctor to patient Track admission status

(iii) Staff Management :

-Add hospital staff View all staff members Manage department and roles

(iv)Appointment System :

-Book appointments View all appointments Update appointment status (Scheduled / Completed / Cancelled)

(v)Billing System :

-Generate patient bills Include consultation, medicine, and room charges View bill receipts Mark bill as paid

(vi)Dashboard :

-Display hospital statistics: Total doctors Total patients Admitted patients Staff count Appointments Bills generated
